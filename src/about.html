The Website

This site has been hand crafted in simple HTML. It has been styled with Tailwind CSS. The files are being served by NGINX on a little GNU/Linux box somewhere in the AWS Cloud. See the GitHub repository here for the source code and more details on the development process.

Some cool features of this site:

  • •List item, Accessible to screen readers, thanks to ARIA.
  • •List item, Uses a large font for easier reading.
  • •List item, Shifts to a dark theme based on system preferences.
  • •List item, Mobile friendly responsive layout.

The Author

I'm Sivasankaran K B, a software developer/engineer by passion and by profession. I've been playing around with computers since 2011. I've worked nearly 5 years in the industry.

Early Life

I was born in a village in Thrissur, Kerala. I still live there. My parents are well educated farmers. They tell me that I used to break my toys, curious about their inner workings. They also tell me I was fond of the English language.

Education

I did my schooling at a CBSE school near home. I was top of my class in Computer Science. I got into HTML, CSS, JavaScript, Java and C/C++ while at school. I also used to write a little hobby x86 Operating System back then.

I have a bachelor's degree in Computer Science and Engineering from a private college near home. While there, I got into Python. I competed in a few local coding contests at college Tech Fests and even won some of them. I also worked at two startups, the second of which turned into my first real job.
